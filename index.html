Data Science Portfolio by Amanda Strack

This portfolio is a compilation of notebooks that explore different machine learning and data analysis techniques. Projects are excecuted both in Python and R.

Springboard Capstone Projects

Playlist Creation Using Audio Features

In this project, I use audio features extracted from the Spotify API to classify songs into a playlist category. Machine learning techniques utilized include Random Forests, SVM, and XGBoost. I also use emsembling method of the three models in the final tuning of the model.

Exploratory Data Analysis

Mobile App Store Analysis

GitHub

This Kaggle dataset contains more than 7000 Apple iOS mobile application details. I explored the app statistics across different genres and how the app details contribute to user ratings.

Dr. Semmelweis and the Discovery of Handwashing

GitHub

In this Datacamp.com project, I analyzed the data behind the breakthrough discovery of handwashing in 1847. I used the basics of Tidyverse (dplyr, ggplot2) to manipulate, analyze and visualize the data.

Who is Drunk and When in Ames, Iowa?

GitHub

This is another project from Datacamp.com. The data contains information on breath alcohol tests in Ames, Iowa, (home of Iowa State University) from 2013-2017. I use tidyverse to explore patterns of breathalyzer tests in this college town.
